# Askora

Askora is a modern, ChatGPT-inspired web platform that enables users to interact with AI in real time. It features a fully functional AI conversation interface with a sleek, intuitive, and user-friendly experience.

---

## Features

- Real-time AI conversation interface  
- Built with **Node.js** and **React** for full-stack functionality  
- **MongoDB** for persistent conversation storage  
- Integrated with **OpenAI API** (knowledge cutoff 2023) for AI responses  
- Syntax highlighting for better code readability in conversations  

---

## Tech Stack

- **Frontend:** React  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **AI:** OpenAI API (cutoff 2023)  
- **Additional Features:** Highlight.js for code syntax highlighting  

---

## Installation
**Clone the repository**  
   ```
   git clone https://github.com/ayushrajgupta01/askora.git
   cd askora
   ```
**Install dependencies**
```
npm install
cd client
npm install
```

**Setup environment variables**
-Create a .env file in the root directory:
```
OPENAI_API_KEY=your_openai_api_key
MONGO_URI=your_mongodb_connection_string
PORT=8080
```

**Start backend:**  
```
npm run server
```

**Start frontend:**
```
cd client
npm start
```
**Access the app**
[Open your browser and go to](http://localhost:5173)

**Usage**
*Chat with Askora in real time
*Code snippets will have syntax highlighting thanks to Highlight.js
*All conversations are stored in MongoDB
