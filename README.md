# Askora

Askora is a modern, ChatGPT-inspired web platform that enables users to interact with AI in real time. It features a fully functional AI conversation interface with a sleek, intuitive, and user-friendly experience.

---

## Features

- Real-time AI conversation interface  
- Built with **Node.js** and **React** for full-stack functionality  
- **MongoDB** for persistent conversation storage  
- Integrated with **OpenAI API** (knowledge cutoff 2023) for AI responses  
- Syntax highlighting for better code readability in conversations  

---

## Tech Stack

- **Frontend:** React  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **AI:** OpenAI API (cutoff 2023)  
- **Additional Features:** Highlight.js for code syntax highlighting
